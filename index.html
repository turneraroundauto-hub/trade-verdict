📊 HOW THIS APP WORKS
Trade Verdict runs the Catalyst Response Framework (CRF) — a 5-gate entry checklist built from 90-day analysis of MU, ASML, and ALAB price/volume/options data. Each gate checks a specific condition before recommending entry.
Gate 0 — Sector gate: SPY + QQQ direction (live data)
Gate 1 — Pre-window exhaustion: 14-day run check (IV-enhanced)
Gate 2 — Catalyst congruence: Canary/Sentiment/Flow classification + put/call skew
Gate 3 — Opening bar: Monday/Friday pattern (67% Friday reversal frequency)
Gate 4 — Phase identification: 1/2/3 cycle position (ATM IV override)
Gate 5 — Korea proxy: KOSPI/Samsung/SK Hynix status (AI/semi names only)
Options overlays (where available): ATM implied volatility, put/call skew, and expected 1-sigma move are fetched from live options chains and used to override AI estimates for Gates 1 and 4.
